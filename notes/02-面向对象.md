# 面向对象编程

面向对象编程（OOP）是一种编程思想，把现实世界中的实体抽象成类和对象，通过封装属性和行为到类中，通过继承复用代码，通过多态实现灵活调用。

## 一、面向对象编程思想

1、面向过程-------步骤

2、面向对象-------角色

例子：我向驾校报名，驾校对我教学

我、驾校-------实体-------对象

我有特征，身高体重等；驾校有牌子，位置价格等------属性和功能

报名，教学----步骤--------实体执行的功能或动作

## 四、魔术方法

在Python中，`__xxx__()`的函数叫做魔法方法，指的是具有**特殊功能**的函数。

1.`__init__()`   初始化对象属性

2`__str__()` 一定要用return  !!!!!!!!!!!

3.`__del__()`释放资源

```python
class Person():
    def __init__(self , name , age):
        self.name = name
        self.age = age
    def __str__(self):
        return f'名字:{self.name}，年龄:{self.age}'
    def __del__(self):
        print(f'{self}删除释放资源')
    
p1=Person('金城武',30)

print(p1.name)
print(p1)
```

## 五、三大特性

封装、继承、多态

（一）封装

属性和方法书写到类的里面的操作即为封装

私有化：在属性名和方法名 前面 加上两个下划线 `__`；一般定义函数名' get_xx '用来获取私有属性，定义' set_xx '用来修改私有属性值。

```python
class Account():
    def __init__(self,name):
        self.name = name
        self.__money = 100
    def get_money(self):
        return self.__money
    def set_money(self,money):
        self.__money = money

user=Account('金城武')
print(user.name)
#print(user.money)
print(user.get_money())
user.set_money(200)
print(user.get_money())
```

4 super()  &  5 MRO

 `super().属性` 或者`super().方法名()`

通过`类名.__mro__`或`类名.mro()`获得“类的层次结构”

```python
class Car(object):
    def __init__(self, brand, model, color):
        self.brand = brand
        self.model = model
        self.color = color

class ElectricCar(Car):
    def __init__(self, brand, model, color, battery):
        super().__init__(brand, model, color)
        # 电池属性
        self.battery = battery

    def run(self):
        print(f'i can run with electric，i has a {self.battery} + "kwh battery"')

print(ElectricCar.__mro__)
print(ElectricCar.mro())
```

(三)  多态

不同对象 => 使用相同方法 => 产生不同的执行结果。

多态依赖继承（不是必须的）

子类方法必须要重写父类方法

```python
class Fruit(object):
    def makejuice(self):
        print('i can make juice')

class Apple(Fruit):
    # 重写父类方法
    def makejuice(self):
        print('i can make apple juice')

class Banana(Fruit):
    # 重写父类方法
    def makejuice(self):
        print('i can make banana juice')

class Orange(Fruit):
    # 重写父类方法
    def makejuice(self):
        print('i can make orange juice')

# 定义一个公共接口（专门用于实现榨汁操作）
def service(obj):
    # obj要求是一个实例化对象，可以传入苹果对象/香蕉对象
    obj.makejuice()

# 调用公共方法
service(Orange())
```



## 六、类属性、类方法、静态方法

属性可以分为**实例属性**和**类属性**

（一）类属性：所有实例对象都有这个属性

```python
class Person(object):
    # 定义类属性
    count = 0
    #方法外部
    def __init__(self, name, age):
        self.name = name
        self.age = age
        
        Person.count+=1

p1 = Person('金城武', 25)
p2 = Person('梁朝伟', 30)
print(f"我们看了帅哥{Person.count}次")
```



（二）类方法

强调数据封装性。不建议直接在类的外部对类属性进行直接获取。

如果想操作类属性，建议使用类方法。

`@classmethod`  and  `def xx_xxx(cls):`

```python
class Person(object):
    # 定义类属性
    count = 0
    #方法外部
    def __init__(self, name, age):
        self.name = name
        self.age = age
        
        Person.count+=1
        
    @classmethod    
    def get_count(cls):
        print(f"我们看了帅哥{cls.count}次")

p1 = Person('金城武', 25)
p2 = Person('梁朝伟', 30)
print(f"我们看了帅哥{Person.count}次")
Person.get_count()
```



（三）静态方法

在开发时，如果需要在类中封装一个方法，这个方法：

① 既 不需要访问实例属性或者调用实例方法

② 也 不需要访问类属性或者调用类方法

这个时候，可以把这个方法封装成一个静态方法

```python
class Person(object):
    @staticmethod
    def game():
        print("打开ps5")
        print("登录账号")
        print("游戏ing")
        
Person.game()
```

面向对象的核心就是：封装属性和行为到类中，通过继承复用代码，通过多态实现灵活调用。
