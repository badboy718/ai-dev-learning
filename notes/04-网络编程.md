一、计算机网络

（一）IP地址

IP地址是分配给网络设备上网使用的数字标签，它能够标识网络中唯一的一台设备，好比现实中每个人都有一个手机号。

IP地址一共有4位，第1位，1~223，第234位，0~255

例如：192.168.88.1

① IP地址分为两类 : IPv4和IPv6

② IPv4是目前使用的IP地址

③ IPv6是未来使用的IP地址

（二）端口号

端口就是传输数据的通道，好比教室的门，是数据传输必经之路。

每一个端口都会有一个对应的端口号

通过ip地址找到对应的设备，通过端口号找到对应的端口，然后通过端口把数据给应用程序。

① 知名端口号：从0到1023

② 动态端口号：从1024到65535

（三）TCP协议

TCP的英文全拼(Transmission Control Protocol)简称传输控制协议，它是一种面向连接的、可靠的、基于字节流的传输层通信协议。

TCP通信步骤：① 创建连接 ② 传输数据 ③ 关闭连接

特点：① 面向连接  ② 可靠传输

二、编码转换

encode = 编码 将字符串转化为字节码

decode = 解码 将字节码转化为字符串

```python
str1="我是奶龙"
str2=str1.encode(encoding="UTF-8")
str3=str2.decode(encoding="UTF-8")
print('str2',str2)
print('str3',str3)
```

输出结果：

str2 b'\xe6\x88\x91\xe6\x98\xaf\xe5\xa5\xb6\xe9\xbe\x99'
str3 我是奶龙	

三、网络编程

（一）socket套接字

计算机之间进行通信的一种约定或一种方式。

只要跟网络相关的应用程序或者软件都使用到了socket。

```python
① 导入socket模块
import socket

② 创建客户端socket对象使用socket类
socket.socket(AddressFamily, Type)
```

(二) TCP客户端和服务器端

客户端:

五步走

1 创建客户端socket对象

2 和服务器端socket连接

3 发送数据

4 接收数据

5 关闭客户端socket

```python
import socket

if __name__ == '__main__':
    # 1、创建客户端套接字对象
    tcp_client_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    # 2、和服务器端套接字建立连接(参数必须是一个元祖)
    tcp_client_socket.connect(("192.168.31.181", 8080))
    # 3、发送数据
    tcp_client_socket.send('hell,itheima'.encode(encoding='utf-8'))
    # 4、接收数据
    recv_data = tcp_client_socket.recv(1024).decode('utf-8')
    print(recv_data)
    # 5、关闭客户端套接字
    tcp_client_socket.close()
```

服务器端：

七步走

1 创建服务器端socket对象

2 绑定IP和端口号

3 设置监听

4 等待客户端的连接请求

5 接收数据

6 发送数据

7 关闭socket

```python
import socket

if __name__ == '__main__':
    # 1、创建服务器端套接字对象
    tcp_server_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    # 2、绑定IP地址与端口号
    tcp_server_socket.bind(("", 8888))
    # 3、开启监听
    tcp_server_socket.listen(128)
    # 4、等待接收客户端连接请求
    conn_socket, ip_port = tcp_server_socket.accept()
    print('客户端IP+端口：', ip_port)
    # 5、接收数据
    recv_data = conn_socket.recv(1024)
    print('接收到的数据：', recv_data.decode())
    # 6、发送数据
    conn_socket.send("客户端的数据已经收到了".encode())
    # 7、关闭套接字
    conn_socket.close()
    tcp_server_socket.close()
```
