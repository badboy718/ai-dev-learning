# 闭包和装饰器

## 一、全局变量和局部变量

① 全局作用域 访问 全局变量，局部作用域 访问 局部变量

② 局部作用域中可以访问全局变量

③ 全局作用域中不能访问局部变量      because  释放资源 弹栈

```python
num=1   #全局变量
def func():
    num2=2
    print(num2)#局部 访问 局部变量
    print(num)#局部 访问 全局变量
print(num)     #全局 访问 全局变量
#print(num2)    #全局 访问 局部变量  报错 ！！！
```



## 二、闭包

闭包 = 嵌套函数 + 引用外部变量 + 返回内部函数

第一步：有嵌套

第二步：有引用

第三步：有返回

```python
def func01():        #第一层函数
    num=100
    def func02():    #嵌套 第二层函数
        print(num)   #引用 调用外部函数的变量
    return func02    #返回  一定要在外层函数返回内部函数
f=func01()
f()
```



## 三、装饰器

在不改变现有函数源代码以及函数调用方式的前提下，实现给函数增加额外的功能。

装饰器的本质就是一个闭包函数（三步：① 有嵌套 ② 有引用 ③ 有返回）

```python
def fn1(fn):                 #fn接收fn3/fn4
    print("fn1")
    def fn2(num1, num2):    
        #num1,num2同步n1,n2  fn2就是fn3/fn4
        print("fn2", num1, num2)
        res=fn(num1, num2)   #接收fn3/fn4的return n1+n2的值
        print(res)
    return fn2               # 返回

@fn1
def fn3(n1,n2):
    print("fn3")
    return n1+n2

@fn1
def fn4(n1,n2):
    print("fn4")
    return n1+n2

fn3(10,20)
fn4(50,100)
```


## 四、通用装饰器模板

```python
def decorator(func):
    def wrapper(*args, **kwargs):
        # 前置功能（执行原函数之前）
        result = func(*args, **kwargs)
        # 后置功能（执行原函数之后）
        return result
    return wrapper

@decorator
def my_function():
    print("原函数执行")
```
