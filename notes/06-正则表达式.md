一、定义

一种用于匹配字符串中字符组合的模式。它由普通字符和特殊字符（元字符）组成，形成了一种强大的文本处理工具。

正则表达式可以用来检查一个大串是否含有某种子串、将匹配的子串做替换或者从某个串中取出符合某个条件的子串等。

① 数据验证（表单验证、如手机、邮箱、IP地址） ② 数据检索（数据检索、数据抓取） ③ 数据隐藏（135****6235 王先生） ④ 数据过滤（敏感关键词过滤） …

`import re`

三、匹配方式

（一）re.match

只能从 **开头** 开始匹配,只能返回 **第一个匹配成功的内容**

匹配成功: 返回匹配对象,还需要从匹配对象中使用**group()**提取数据
匹配失败: 返回None

```python
import re

result = re.match('123','12345 123')
#返回123 只能匹配第一个123内容
#result = re.match('123','45 12345 123') 
# no match 开头不是123

if result:
    print(result.group())
else:
    print('no match')
```

（二）re.search

在整个大字符串中查找,只返回第一个匹配成功的数据

匹配成功: 从返回的数据对象中把数据取出来，使用**group()**提取数据

匹配失败: 返回None

```python
import re

result = re.search('123','45 12345 123')
# 返回123 整段中找，找出第一个
if result:
    print(result.group())
else:
    print('no match')
```

（三）re.findall

可以整个字符串搜索,可以返回所有匹配成功的内容

匹配成功: 返回的包含所有匹配成功数据的列表->[元素1,元素2,元素3...]

不需要**group()**提取数据

匹配失败: 返回空列表->[]

```python
import re

result=re.findall('123','45 123 45 123 12345')
# 返回 列表 ['123', '123', '123']
if result:
    print(result)
else:
    print('no match')
```

四、匹配规则

（一）匹配单个字符

```python
# 记忆方式:  d:digit数字  s:space空白   w:word词
#
# . : 任意1个字符(除了\n)
# []: 中括号中任意1个字符
#
# \d: 任意1个数字(0-9)
# \D: 非数字
#
# \s: 任意1个空白(空格,\t,\n)
# \S: 非空白
#
# \w: 任意1个正常词(字母,数字,下划线,汉字)
# \W: 非正常词(特殊字符)

import re

def show(result):
    if result:
        print(result.group())
    else:
        print("No match")

short_str='Abc123'
long_str='Xy_7-你好 2001-02-03 a@b.com 127.0.0.1 13070101975 -9.9 !@#\t\n'

# . 任意一个字符
result=re.match('.b..',short_str)
show(result)

# [] []中任意一个字符
result=re.match('[ABC][a-z][a-d][0-9][1-4][0-5]',short_str)
show(result)

# \d 任意一个数字
result=re.match('...\d\d[0-4]',short_str)
show(result)

#\D 任意一个非数字
result=re.match('\D\D\D\d\d\d',short_str)
show(result)

#\s 任意一个空白
result=re.match('\D..\s.\d.','Abc 123')
show(result)

#\S 任意非空白
result=re.match('\D\S.\s.\d\S','Abc 123')
show(result)

#\w 任意正常字（字母、数字、下划线、汉字）
result=re.match('\D\S\w\s.\d\S','Abc 123')
show(result)

#\W 任意非正常词（特殊字符）
result=re.match('\W\D\S\w\s.\d\S','@Abc 123')
show(result)
```

（二）匹配多个字符

```python
# * : 匹配前一个字符出现0次或者无限次     次数>=0
#
# + : 匹配前一个字符出现1次或者无限次     次数>=1
#
# ? : 匹配前一个字符出现0次或者1次       次数== 0 或者 次数 == 1
#
# {x}: 匹配前一个字符出现x次            次数==x
#
# {x,y}:匹配前一个字符出现x到y次        次数>=x 并且 次数<=y

import re

def show(result):
    if result:
        print(result.group())
    else:
        print("No match")

# * : 匹配前一个字符出现0次或者无限次     次数>=0
result=re.match('[A-C]*.\D[c]*\d*','Abbccc')
show(result)

# + :匹配前一个字符出现1次或者无限次      次数>=1
result=re.match('[A-C]*.\D[c]+\d*','Abbccc')
show(result)

# ? :匹配前一个字符出现0次或者1次         次数==0 或者 ==1
result=re.match('[A-C]?.\D[c]?[c]+\d*','Abbccc')
show(result)

# {x}:匹配前一个字符出现x次              次数==x
result=re.match('[A]{1}[b]{2}[c]{3}','Abbccc')
show(result)

# {x,y}:匹配前一个字符出现x到y次         次数>=x and 次数<=y
result=re.match('[A-C]{0,3}[b]{0,3}[c]{0,3}','Abbccc')
show(result)
```

（三）匹配开头、结尾

```python
# ^ : 匹配以什么开头
# $ : 匹配以什么结尾
# 注意: ^和$一般都是配合使用,起到限制作用

import re

def show(result):
    if result:
        print(result.group())
    else:
        print("No match")

# ^ :匹配以什么开头
result=re.match('^A','Abc_123')
show(result)

# $ :匹配以什么结尾  $写在最后
result=re.match('^A.....3$','Abc_123')
show(result)
```
